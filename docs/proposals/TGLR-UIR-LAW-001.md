## Symbol definitions

- \(\mathcal U_0\): initialized finite base state set for bounded demonstrations.
- \(\Phi_{\mathcal I}\): informational transition operator.
- \(\Phi_{\mathcal I}^{\,n}\): n-fold application of \(\Phi_{\mathcal I}\).
- \(\tau\): topology used for closure.
- \(\operatorname{cl}_{\tau}(\cdot)\): closure under topology \(\tau\).
- \(\mathcal R\): informational reality set induced by iterative expansion.
- \(\subsetneq\): strict subset relation (strict growth).
- \(x\leftrightsquigarrow_{\mathcal I}^{*}y\): informational-path connectivity between states.

## Boundary between axiom and executable proxies

- The UIR expression above is a proposed infinite research axiom.
- Repository software executes only finite bounded proxies.
- Finite tests (for example 100-state growth/connectivity proxies) are demonstration evidence only.
- Finite tests do not prove infinity.
